# lilMargin

Margin and Markup Calculator

Turn cost into price the right way. Enter your cost and any one of margin, markup, or sale price, and lilMargin works out the rest, with a plain explanation of why margin and markup are not the same number.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
